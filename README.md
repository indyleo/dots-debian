# Awesome loading ~/.xsession
In /usr/share/xsessions/awesome.desktop change the exec line to
```
Exec=/etc/X11/Xsession
```

# Qt And GTK Themes
```
QT_QPA_PLATFORMTHEME=qt5ct 
```
Put this into your '/etc/environment' file
